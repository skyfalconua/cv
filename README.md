# 📄 Andrew Sokolov - CV

Git-friendly version of my CV built with [Astro](https://astro.build).

## 🚀 Run Locally

```bash
npm i
npm run dev
# Open `http://localhost:4321` and save as PDF (Ctrl+P / Cmd+P)

npm run build
# Result in dist/index.html
```
